# TannerForGA

Static campaign website for Tanner for Georgia State House District 20.

## Tech
- Plain HTML + JavaScript (no server-side components)
- Static hosting friendly (GitHub Pages / Netlify / S3)

## Getting started
- Open `index.html` directly in your browser, or run a local static server.
- Example: `python3 -m http.server 8080` then visit `http://localhost:8080`.

## Preview
- This repo includes a GitHub Actions workflow that deploys to GitHub Pages on every push to `main`.
- Once enabled in GitHub repo settings, your preview URL will be: `https://<org-or-user>.github.io/<repo>/`.

## Notes
- Keep the repo static-only (no backend).
